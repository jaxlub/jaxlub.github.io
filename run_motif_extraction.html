run_motif_extraction {tools}R Documentation

Motif Extraction

Description

Reformat data file, build Blast Database, pull sequences and add output.

Usage

run_motif_extraction(input_path, output_path, ncbi_bin, protein_ref, num = 7)

Arguments

input_path

Path to input file

output_path

Path to output file

ncbi_bin

Path to ncbi-blast-2.14.0+/bin

protein_ref

Path to Fasta refrence file

num

Number of amino acids to extract on either side of the modified amino acid (default = 7)

Value

Input file with addition of protein sequences
