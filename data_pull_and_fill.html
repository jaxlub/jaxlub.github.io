data_pull_and_fill {tools}R Documentation

Fills in Sequences of input file

Description

Loops through reformatted input file and pulls 15 character ranges from blast.

Usage

data_pull_and_fill(input_file, ref_protein, ncbi_bin, num = 7)

Arguments

input_file

Input data file

ref_protein

Path to Fasta refrence file

ncbi_bin

Path to ncbi-blast-2.14.0+/bin

num

Number of amino acids to extract on either side of the modified amino acid (default = 7)

Value

Final Data Sheet w/ Extracted Sequences
