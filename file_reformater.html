file_reformater {tools}R Documentation

Format File Input

Description

Parse and reformat input file to have only 1 row corresponding to 1 accession.

Usage

file_reformater(input_path, num = 7)

Arguments

input_path

Path to input file

num

Number of amino acids to extract on either side of the modified amino acid (default = 7)

Value

Reformatted Input File
